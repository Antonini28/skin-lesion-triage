# SkinSense — Presentation & Reports

Supporting material for the SkinSense project. SkinSense combines two AI systems: a
computer-vision **triage model** and **DermBot**, a conversational clinical **LLM** assistant
(Gemini 2.5 Flash + RAG). Both are covered in the material below.

## Slides (`slides/`)
- **SkinSense_AI_Conference.pptx** — full conference presentation of the system, model, and DermBot.
- **SkinSense_AI_Vlog.pptx** — shorter vlog / demo deck.

## Reports (`reports/`)
- **Horizon_Scan_Report_DermBot_FINAL.docx** — final horizon-scan report covering the DermBot LLM.
- **Horizon_Scan_Report_DermBot.md** — markdown version of the DermBot report.
- **Horizon_Scan_Report_BCU_Harvard.docx** — horizon-scan report (BCU / Harvard framing).
- **DOCBOT_IMPLEMENTATION_PLAN.md** — the DermBot (LLM) implementation plan: RAG, safety
  alignment, and integration into the triage app.

> The decks and reports were authored during development; some still use the earlier
> "SkinTriage" name for what is now branded **SkinSense**.
